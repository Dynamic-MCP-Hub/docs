---
title: "List Overrides"
api: "GET /api/rate-limits/overrides"
description: "List all client rate limit overrides (admin only)"
---

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token: `Bearer <admin_token>`
</ParamField>

## Query Parameters

<ParamField query="active_only" type="boolean" default="false">
  Only return active overrides
</ParamField>

### Example Request

```bash
curl "https://api.sari-platform.com/api/rate-limits/overrides?active_only=true" \
  -H "Authorization: Bearer <admin_token>"
```

## Response

```json
[
  {
    "id": 1,
    "client_id": "client-uuid",
    "allowed_limit": 5000,
    "window_seconds": 3600,
    "is_active": true,
    "created_at": "2024-01-15T10:00:00Z",
    "updated_at": "2024-01-15T10:00:00Z"
  }
]
```

## Errors

| Status | Description |
|--------|-------------|
| 401 | Invalid or expired token |
| 403 | Admin access required |
