## Navigate with Intelligence

Sari is a comprehensive API management platform that enables AI agents to discover and execute APIs through natural language. It provides:

- **Dynamic API Discovery** - Semantic search across your API collections
- **Secure Access Control** - Fine-grained permissions per client and collection
- **MCP Integration** - Native support for Model Context Protocol
- **Admin Console** - Web-based management interface

## Key Features

<AccordionGroup>
  <Accordion title="Semantic API Search" icon="magnifying-glass">
    Use natural language to find relevant APIs. Powered by Vertex AI embeddings and pgvector for fast similarity search.

    ```
    Query: "payment processing API"
    Results:
    1. Stripe Payment API (similarity: 0.89)
    2. PayPal Checkout API (similarity: 0.82)
    ```
  </Accordion>

  <Accordion title="Dual Authentication" icon="shield-check">
    Support for both Firebase Authentication (web users) and API Key authentication (AI agents).

    - **Firebase**: `Authorization: Bearer <id_token>`
    - **API Key**: `X-Client-ID` + `X-API-Key` headers
  </Accordion>

  <Accordion title="Rate Limiting" icon="gauge">
    Configurable rate limits with tier-based defaults and per-client overrides. Redis-backed for high performance.
  </Accordion>

  <Accordion title="Audit Logging" icon="clipboard-list">
    Complete audit trail of all API interactions for compliance and debugging.
  </Accordion>
</AccordionGroup>
